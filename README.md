# `Clash#`

[English](./README.md)

![Clash# Logo](./Logo.png)

`Clash#` is a modern, Windows-native proxy client based on [mihomo](https://github.com/MetaCubeX/mihomo).  
Download it from [GitHub Releases](https://github.com/Water-Run/ClashSharp/releases), then extract the package and run `Installer.exe` to install.  
`Clash#` requires `Windows 11` to run. It is open-source on [GitHub](https://github.com/Water-Run/ClashSharp) under the `AGPL 3.0` license.
